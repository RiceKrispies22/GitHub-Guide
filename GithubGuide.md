# Github guide
I will be talking about how to import your code from VS code into GitHub.

# git init 

``` This starts a brand new Github repository in your folder so Github can begin tracking your files that you upload```

# git add README.md
```This tells Github, “I want to include this file in my next respatory.” Basically, you’re marking it as ready to be saved.```

# git commit -m "first commit"
``` This saves your changes with a message explaining what you did. A commit is like a checkpoint of your project.
```
# git branch -M main
``` This renames your current branch to main, which is the standard default branch name on GitHub.```

# git remote add origin https://github.com/RiceKrispies22/GitHub-Guide.git
```This renames your current branch to main which is the defult branch name on GitHub.```

#git push -u origin main
``` This connects your project to the GitHub repository you created online. It tells Git where to send your code.```
